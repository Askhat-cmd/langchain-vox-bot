## 🎯 ОБЗОР ПРОЕКТА

### **Что это такое:**
Голосовой AI-ассистент для компании "Метротэст", который:
- **Принимает звонки** через Asterisk ARI (телефонная система)
- **Понимает речь** клиента с помощью Yandex ASR
- **Находит ответы** через AI-агента с LangChain и RAG (Retrieval-Augmented Generation)
- **Отвечает вслух** используя Yandex TTS (Text-to-Speech)
- **Поддерживает barge-in** - прерывание бота при речи клиента
- **Предоставляет веб-интерфейс** для администрирования

### **Ключевые возможности:**
- 🎙️ **Голосовое взаимодействие** - полный цикл: речь → понимание → ответ → воспроизведение
- 🧠 **Две базы знаний** - general (общие вопросы) и tech (технические)
- ⚡ **Оптимизированная производительность** - 3.4x ускорение (9.5с → 2.9с)
- 🔄 **Redis кеширование** - ускорение повторных запросов
- 📊 **Веб-админка** - мониторинг, управление, аналитика
- 🛡️ **Отказоустойчивость** - fallback механизмы на всех уровнях

## 🏗️ АРХИТЕКТУРА СИСТЕМЫ

### **Основные компоненты:**

| Компонент | Файл | Назначение |
|-----------|------|------------|
| **Stasis Handler** | `app/backend/asterisk/stasis_handler_optimized.py` | 🎯 **ГЛАВНЫЙ ОБРАБОТЧИК** - управляет всем циклом звонка |

### **⚙️ ТЕХНИЧЕСКИЕ ДЕТАЛИ АВТООБНОВЛЕНИЯ IAM ТОКЕНОВ:**

**В `yandex_asr_service.py`:**
- `_get_iam_token()` - проверяет и обновляет токен при необходимости
- `token_needs_refresh()` - проверяет время истечения (за 30 минут)
- `_refresh_iam_token()` - получает новый IAM токен из OAuth токена
- **Логирование:** `✅ IAM токен обновлен! Действует до: 2025-09-17 22:18:33`

**Алгоритм работы:**
1. При каждом запросе ASR вызывается `_get_iam_token()`
2. Проверяется `token_needs_refresh()` - нужно ли обновление
3. Если да - вызывается `_refresh_iam_token()` для получения нового токена
4. Новый токен действует 12 часов
5. При ошибке 401 - принудительное обновление и повторный запрос
